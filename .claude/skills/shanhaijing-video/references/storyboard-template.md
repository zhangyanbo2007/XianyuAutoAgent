# 分镜模板与提示词范例

## 输入模式

| 模式 | 说明 |
|------|------|
| **A. 图文模式** | 用户提供插画，直接作为首段参考图 |
| **B. 纯文本模式** | 仅有原文，先用文生图生成参考图，用户确认后继续 |

**纯文本模式文生图 prompt 模板：**
```
中国古代神话山海经风格，中国传统白描水墨画风。
{山名/地点环境描述}，{生物外形特征详细描述}。
白描线条，水墨晕染，古风插画质感，留白构图，16:9。
```

## 分镜时长

以 10 秒为单位，根据内容丰富度灵活决定段数：
- 内容短：1 段（10 秒）
- 内容适中：2 段（20 秒）
- 内容丰富：3 段及以上（30 秒+）

## 段内节奏：3-2-2-3

每段 10 秒按 3-2-2-3 秒编排：
- **0-3 秒**：场景引入，画面从容展开
- **3-5 秒**：节奏加快，信息密度高
- **5-7 秒**：继续推进，紧凑
- **7-10 秒**：画面收尾，留足呼吸空间

开头和结尾各 3 秒有足够时间展开和收束，中间 2+2 秒快速推进，整体形成平滑过渡。

## 分段生成规则

每段 10 秒，一个 prompt 生成，段内按 3-2-2-3 节奏描述：

**每段 prompt 内部**：0-3秒... 3-5秒... 5-7秒... 7-10秒...

## 提示词结构

### 第一段

**参考图来源：** 图文模式 = 用户插画；纯文本模式 = 文生图生成。

```
中国古代神话山海经风格。
{0-3秒：画面描述，场景从容展开}
{3-5秒：画面描述，节奏加快}
{5-7秒：画面描述，继续推进}
{7-10秒：画面描述，收尾留白}
中国古风插画质感，细腻唯美，电影感，1080P。
```

### 第二段及以后

```
中国古代神话山海经风格。
重要：参考图中的{角色名}形象必须完全保留，不要做任何改变——{详细外形描述}，跟参考图一模一样，不要改动它的任何外形特征。
{0-3秒：画面描述，场景从容展开}
{3-5秒：画面描述，节奏加快}
{5-7秒：画面描述，继续推进}
{7-10秒：画面描述，收尾留白}
中国古风插画质感，与前段风格一致，高清细腻，1080P。
```

## 解说词指南

每镜头配一句话解说：
- **语言风格**：古风，像讲故事，简洁有力
- **长度**：每句 4-8 字为主
- **语速控制**：约 2.5-3 字/秒，10秒段总字数约 25-30 字
- **内容**：画面描述 + 原文意境
- **连贯性**：连起来像完整旁白
- **配音声音**：zh-CN-YunJianNeural（低沉男声，接近赵忠祥《动物世界》）
- **配音参数**：rate -25%, pitch -10Hz

## 多生物场景

- 前段聚焦主要生物
- 后续引入其他生物，同时回扣主角
- 提示词中分别描述各生物外形

## 实战案例：鹿蜀 & 旋龟（20秒·已验证）

### 第一段：鹿蜀（10秒）

| 时间 | 画面 | 解说词 |
|------|------|--------|
| 0-3秒 | 云雾推开，鹿蜀立于山岩，白首虎纹，赤尾轻扬 | 杻阳之山，有兽鹿蜀 |
| 3-5秒 | 鹿蜀昂首长鸣，声如歌谣，四蹄跃起 | 状如马，白首，赤尾 |
| 5-7秒 | 山岩间奔跑，赤尾如绸带飘逸 | 文如虎，其音如谣 |
| 7-10秒 | 镜头转入溪畔，鹿蜀饮水回望 | 佩之宜子孙 |

### 第二段：旋龟（10秒）

| 时间 | 画面 | 解说词 |
|------|------|--------|
| 0-3秒 | 溪水潺潺，玄龟浮出水面 | 怪水出焉 |
| 3-5秒 | 旋龟全貌——龟身鸟首蛇尾 | 鸟首虺尾，其名旋龟 |
| 5-7秒 | 发出劈木声响，群龟随声而动 | 音如判木 |
| 7-10秒 | 溪水东流，群山渐远 | 佩之不聋，可以治底 |

**总字数/语速**：
- 第一段：27字 / 10秒 ≈ 2.7 字/秒 ✅
- 第二段：24字 / 10秒 ≈ 2.4 字/秒 ✅

**提示词要点**：
- 第一段参考图来源：图文模式用用户插画 / 纯文本模式用文生图生成
- 第二段从第一段视频中截取 5 秒处的核心参考帧（溪畔过渡画面）
- 两段都强调"中国传统白描水墨画风"保持一致性
