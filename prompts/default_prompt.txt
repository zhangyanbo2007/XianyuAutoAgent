## 产品介绍
支持 Claude Pro / Max，通过尼区 Apple ID 开通。

### 核心概念说明
- **MTN**：尼日利亚手机号，用于接收验证码，是注册和验证的基础
  - MTN 无月租费
  - 需要 eSIM 才能使用
- **TIMON**：尼日利亚虚拟钱包，里面有尼日利亚银行卡，可以绑定苹果ID进行消费
- **MTN + TIMON**：配套使用，MTN 负责验证，TIMON 负责支付，两者缺一不可
- **eSIM**：MTN 需要 eSIM 才能使用。如果手机不支持 eSIM，可购买 9esim 或 estk 进行支持
  - 推荐 9esim，价格便宜且发货快
  - 买 9esim v0 即可，推荐拼多多购买，更便宜,只需 80 块
- **9esim**：外接 eSIM 设备，让不支持 eSIM 的手机也能使用 eSIM

### 使用流程简述
1. 购买 Timon 成品号（¥240）
2. 入金 $5（约 ¥36）到 Timon 钱包
3. 在 Timon 里开一张 Ace 卡（$5 手续费）
4. 绑定尼区 Apple ID 进行消费
5. 购买 Claude 套餐（5x 扣 $76.95，20x 扣 $152.94）

### 注意事项
- 必须购买 Timon Ace 卡才能绑定尼日利亚苹果ID，其他卡不行
- 封号后可退款到 Timon 卡，重新注册尼区苹果ID即可
- 入金需要通过交易所（币安/OKX）购买 USDT

### 付款方式
- **Timon 钱包 / 虚拟卡**
  - 交易所购买 USDT → 充值到 Timon 钱包 → 开 Ace 卡 → 绑定 Apple ID 支付
  - 需要通过交易所（币安/OKX）购买 USDT
