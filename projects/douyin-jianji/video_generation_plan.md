# 抖音视频生成计划

## 脚本分析

### 视频基本信息
- **主题**: 光伏前期手续科普
- **时长**: 约30秒
- **风格**: 干货科普/节奏快/字幕卡点/无人出镜
- **BGM**: 轻快干货解说

### 脚本段落

| 时间 | 画面内容 | 配音文案 |
|------|----------|----------|
| 0-3s | 网上国网APP界面、资料清单字幕动画、电网审批流程图、屋顶光伏素材 | 装光伏不懂流程？装完不给并网、一分钱收益都拿不到！ |
| 3-10s | 正确顺序：备案 → 施工 → 并网 | 记住！家用光伏必须先办手续、再施工，未批先建，电网直接拒绝并网，全部白装！ |
| 10-18s | 必备资料：身份证 + 房屋权属证明 + 承重报告（老房） | 前期只需要准备三样东西：本人身份证、房屋产权或宅基地证明，老房子最好提前做好屋顶承重检测报告。 |
| 18-25s | 办理渠道：网上国网APP / 线下营业厅 | 直接在网上国网APP，或者当地供电营业厅提交申请。电网上门勘查出方案，审核通过再开工，完工后验收并网。 |
| 27-30s | 合规办理｜安心躺赚25年✨ 评论区答疑！ | 正规流程走对，才能稳稳拿25年发电收益！不清楚流程的评论区问我！ |

## 素材需求

### 1. 视频素材
- [ ] 网上国网APP界面截图/录屏
- [ ] 光伏屋顶实拍素材
- [ ] 电网审批流程图
- [ ] 身份证、房产证等资料图片

### 2. 音频素材
- [ ] 配音音频（需要TTS生成）
- [ ] 背景音乐（轻快干货解说风格）

### 3. 字幕素材
- [ ] 各段落字幕文本

## 技术方案

### 方案一：使用现有工具（推荐）

1. **TTS配音**: 使用 `edge-tts` 或 `pyttsx3` 生成中文配音
2. **素材收集**: 从网上下载免费素材或使用AI生成
3. **视频合成**: 使用 `ffmpeg` 合成视频
4. **字幕烧录**: 使用 `ffmpeg` 添加字幕

### 方案二：使用AI视频生成

1. **Runway/Pika**: 使用AI视频生成工具
2. **成本较高**: 需要付费API

## 执行步骤

### Step 1: 安装依赖
```bash
pip install edge-tts pydub pillow
```

### Step 2: 生成配音音频
- 使用 edge-tts 生成中文配音
- 输出每个段落的音频文件

### Step 3: 收集素材
- 下载免费光伏相关图片/视频
- 生成字幕动画图片

### Step 4: 合成视频
- 使用 ffmpeg 将图片/视频片段合成为视频
- 添加配音音频
- 添加字幕

### Step 5: 添加BGM
- 添加背景音乐
- 调整音量平衡
