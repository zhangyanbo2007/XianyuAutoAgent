TraceScope：通过解耦清单裁决实现交互式URL分类现代网络钓鱼活动越来越多地通过使用交互式门（例如，复选框/滑块挑战）、延迟内容渲染和无标志凭证收集器来规避基于快照的URL分类器。这将URL分类从静态分类转向交互式取证任务：分析师必须主动浏览页面，同时将自己与潜在的运行时漏洞隔离开来。我们提出了TraceScope，一个解耦的分类管道，能够大规模地实现这一工作流程。为了防止观察者效应并确保安全，一个沙盒化的操作代理驱动一个真实的GUI浏览器，在视觉动机的引导下引发页面行为，并将会话冻结成一个不可变的证据包。另外，一个裁决代理通过按需查询证据来验证MITRE ATT&CK清单，从而规避了LLM上下文限制，并生成一份包含提取的危害指标（IOCs）和最终裁决的审计就绪报告。在来自现有数据集的708个可访问URL（其中241个是来自PhishTank的已验证网络钓鱼URL，467个是来自Tranco爬取的良性URL）上进行评估，TraceScope实现了0.94的精确率和0.78的召回率，显著提高了相对于三个现有视觉/基于引用的分类器的召回率，同时生成了可复现的、分析师级别的、适合审查的证据。更重要的是，我们手动整理了一个真实世界网络钓鱼电子邮件数据集，以在实际环境中评估我们的系统。我们的评估表明，TraceScope在真实世界场景中也表现出卓越的性能，成功检测出当前最先进防御系统未能识别的复杂网络钓鱼尝试。

原始摘要

Modern phishing campaigns increasingly evade snapshotbased URL classifiers using interaction gates (e.g., checkbox/slider challenges), delayed content rendering, and logoless credential harvesters. This shifts URL triage from static classification toward an interactive forensics task: an analyst must actively navigate the page while isolating themselves from potential runtime exploits. We present TraceScope, a decoupled triage pipeline that operationalizes this workflow at scale. To prevent the observer effect and ensure safety, a sandboxed operator agent drives a real GUI browser guided by visual motivation to elicit page behavior, freezing the session into an immutable evidence bundle. Separately, an adjudicator agent circumvents LLM context limitations by querying evidence on demand to verify a MITRE ATT&CK checklist, and generates an audit-ready report with extracted indicators of compromise (IOCs) and a final verdict. Evaluated on 708 reachable URLs from existing dataset (241 verified phishing from PhishTank and 467 benign from Tranco-derived crawling), TraceScope achieves 0.94 precision and 0.78 recall, substantially improving recall over three prior visual/reference-based classifiers while producing reproducible, analyst-grade evidence suitable for review. More importantly, we manually curated a dataset of real-world phishing emails to evaluate our system in a practical setting. Our evaluation reveals that TraceScope demonstrates superior performance in a real-world scenario as well, successfully detecting sophisticated phishing attempts that current state-of-theart defenses fail to identify.
