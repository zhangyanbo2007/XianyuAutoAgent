原始摘要与中文对照

中文对照翻译

在现实世界中，智能体在漫长且不断演变的时间跨度内运行，信息会反复更新并可能在记忆之间产生干扰，这要求智能体能够准确回忆并对多条信息进行聚合推理。然而，现有基准侧重于静态、独立的记忆回忆，未能捕捉到不断演变的记忆之间的动态交互。在本文中，我们研究了当前记忆增强型智能体在现实、干扰严重、长时程设置中，跨不同领域和问题类型的表现。为此，我们引入了MINTE VAL（长时程干扰下记忆评估），这是一个分析性基准，其特点包括：(1) 具有频繁更新信息且高度互联的长上下文，会产生大量干扰；(2) 多样化的领域（状态跟踪、多轮对话、维基百科修订和GitHub提交），能够评估领域泛化能力；以及 (3) 多样化的问答类型，用于评估对干扰的鲁棒性，包括 (i) 需要从长上下文中检索特定目标的单目标回忆任务，以及 (ii) 需要对多条相关信息进行推理的多目标聚合任务。总体而言，MINTE VAL包含15.6k个问答对，这些问答对基于平均138.8k个token、最长可达1.8M个token的长时程上下文。我们评估了七个代表性系统，包括普通的长期上下文LLMs、检索增强生成方法和记忆增强型智能体框架。在所有系统中，我们观察到性能普遍较低（平均准确率为27.9%），尤其是在需要对多条证据进行聚合推理的问题上。细粒度分析表明，性能主要受限于检索和记忆构建能力。此外，当前的记忆系统难以回忆和推理那些后来被后续上下文修改或干扰的早期事实，并且随着中间更新次数的增加，性能会下降。这些发现强调了在动态、长时程、跨不同领域环境中，需要更强大的记忆管理系统。代码和数据可在https://github.com/amy-hyunji/MINTEval获取。

原始摘要

Agents in real-world settings operate over long and evolving horizons, where information is repeatedly updated and may interfere across memories, requiring accurate recall and aggregated reasoning over multiple pieces of information. However, existing benchmarks focus on static, independent recall and fail to capture these dynamic interactions between evolving memories. In this paper, we study how current memory-augmented agents perform in realistic, interference-heavy, long-horizon settings across diverse domains and question types. To this end, we introduce MINTE VAL (Long-Horizon Memory under INTerference Evaluation), an analytical benchmark which features (1) long, highly interconnected contexts with frequently updated information that induces substantial interference, (2) diverse domains (state tracking, multi-turn dialogue, Wikipedia revisions, and GitHub commits), enabling evaluation of domain generalization, and (3) diverse question types that assess robustness to interference, including (i) single-target recall tasks requiring retrieval of a specific target from long contexts, and (ii) multi-target aggregation tasks requiring reasoning over multiple relevant pieces of information. Overall, MINTE VAL contains 15.6k question-answering pairs over long-horizon contexts averaging 138.8k tokens and extending up to 1.8M tokens per instance. We evaluate seven representative systems, including vanilla long-context LLMs, retrievalaugmented generation methods, and memory-augmented agent frameworks. Across all systems, we observe consistently low performance (avg. 27.9% accuracy), especially on questions requiring aggregated reasoning over multiple pieces of evidence. Fine-grained analysis shows that performance is primarily limited by retrieval and memory construction capabilities. Furthermore, current memory systems struggle to recall and reason over earlier facts that are later revised or interfered with by subsequent context, with performance degrading as the number of intervening updates increases. These findings highlight the need for more robust memory management systems for dynamic, long-horizon environments across varying domains. Code and data are available at https://github.com/amy-hyunji/MINTEval.
