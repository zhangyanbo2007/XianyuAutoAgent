核心要点

问题/背景
自动驾驶里的 Vision-Language-Action 模型正在从显式文本 CoT 走向 latent reasoning，但标准 latent CoT 往往缺乏物理约束，容易形成语义与感知脱节的问题。本文关注的正是这个瓶颈：如何让 VLA 的“思考空间”真正带有时空和动力学结构。
方法/机制
作者提出 LaST-VLA，把推理从离散符号空间移到 latent spatio-temporal space，并通过双特征对齐，把 3D foundation model 的几何约束和 world model 的动态前瞻性蒸馏进 latent 推理空间。训练上先做 progressive SFT，再用 GRPO 强化安全与规则遵循。摘要中给出的结果是在 NAVSIM v1/v2 上达到新的高分。
结果/证据
这篇论文适合仓库的多模态 / world model / VLA 主线，也和 PerpetualWonder、World-VLA-Loop、Drive-JEPA 这一批条目互相补强。它代表的是自动驾驶里“latent spatio-temporal reasoning” 正在替代文本 CoT 的趋势。
收录价值
它目前归为突破性而不是更高一级，原因是方向很强，但仍集中在自动驾驶 VLA 这一特定赛道，且证据主要来自 arXiv 与 benchmark 结果，尚未外溢到更广的通用 VLA 路线。
完整收录解读

自动驾驶里的 Vision-Language-Action 模型正在从显式文本 CoT 走向 latent reasoning，但标准 latent CoT 往往缺乏物理约束，容易形成语义与感知脱节的问题。本文关注的正是这个瓶颈：如何让 VLA 的“思考空间”真正带有时空和动力学结构。

作者提出 LaST-VLA，把推理从离散符号空间移到 latent spatio-temporal space，并通过双特征对齐，把 3D foundation model 的几何约束和 world model 的动态前瞻性蒸馏进 latent 推理空间。训练上先做 progressive SFT，再用 GRPO 强化安全与规则遵循。摘要中给出的结果是在 NAVSIM v1/v2 上达到新的高分。

这篇论文适合仓库的多模态 / world model / VLA 主线，也和 PerpetualWonder、World-VLA-Loop、Drive-JEPA 这一批条目互相补强。它代表的是自动驾驶里“latent spatio-temporal reasoning” 正在替代文本 CoT 的趋势。

它目前归为突破性而不是更高一级，原因是方向很强，但仍集中在自动驾驶 VLA 这一特定赛道，且证据主要来自 arXiv 与 benchmark 结果，尚未外溢到更广的通用 VLA 路线。

中文对照翻译

通过统一感知和规划，彻底改变了自动驾驶，但它们对显式文本思维链 (CoT) 的依赖导致了语义-感知解耦和感知-符号冲突。最近转向潜在推理的趋势试图通过在连续隐空间中思考来规避这些瓶颈。然而，在没有显式中间约束的情况下，标准潜在思维链通常作为一种物理无关表示运行。为解决此问题，我们提出了潜在时空VLA (LaST-VLA)，这是一个将推理范式从离散符号处理转变为物理接地潜在时空思维链的框架。通过实施双特征对齐机制，我们将3D基础模型中的几何约束和世界模型中的动态预见直接提炼到潜在空间中。结合从特征对齐到轨迹生成的渐进式SFT训练策略，并通过基于组相对策略优化 (GRPO) 的强化学习进行优化，以确保安全性和规则遵从性。图1. VLA范式比较。(a) 直接VLA高效但缺乏推理。(b) 显式文本思维链可解释但存在高延迟和幻觉。(c) 朴素潜在思维链 (无监督) 高效但不稳定 (模型崩溃)。(d) 我们的时空潜在思维链 (有监督) 将潜在特征与物理先验对齐，实现了效率、稳定性和接地性。LaST-VLA在NAVSIM v1 (91.3 PDMS) 和NAVSIM v2 (87.1 EPDMS) 上创造了新纪录，同时在SURDS和NuDynamics基准测试中表现出卓越的时空推理能力。代码可在LaST-VLA Code获取。尽管模块化方法提供了可解释性，但由于僵硬的基于规则的设计和模块间的错误传播，它们在复杂和长尾场景中常常表现不佳。相比之下，VLA模型将视觉感知和驾驶策略统一到一个整体系统中，展示了卓越的场景泛化能力和自然语言指令遵循能力。尽管VLA模型已展示出超越传统端到端 (E2E) 方法的有前景的性能。
