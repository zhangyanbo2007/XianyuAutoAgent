完整收录解读

This position paper reframes agent security away from model robustness alone and toward system-level security invariants around tools, identity, authority, memory, and execution boundaries.

Its central claim is that the model inside an agent should be treated as an untrusted component, with security guarantees enforced by surrounding systems mechanisms rather than by hoping the model refuses bad instructions.

The paper analyzes representative real-world agent attacks through this lens and maps them to classical systems-security principles such as isolation, least privilege, mediation, and auditable control boundaries.

For this repository, the value is the reusable threat-model shift: secure agents need operating-system-like boundaries, not only alignment tuning or prompt-level defenses.

原始摘要与中文对照

中文对照翻译

我们认为智能体安全必须作为一个系统问题来处理：为智能体提供动力的AI模型必须被视为一个不可信组件，并且安全不变量必须在系统层面强制执行。从这个角度来看，旨在提高模型鲁棒性（社区中的主流观点）的努力本身是不够的。相反，我们必须用系统安全领域的技术来补充现有工作。基于我们作为操作系统、网络、形式化方法和对抗性机器学习领域的网络安全研究人员的经验，我们阐明了一套核心原则，这些原则植根于数十年的系统安全研究，为设计具有可预测保证的智能体系统提供了基础。作为证据，我们分析了对智能体的十一次代表性真实世界攻击，并讨论了如果实现系统原则，本可以如何阻止这些攻击。我们还指出了在智能体中实现这些原则所面临的研究挑战。

原始摘要

We take the position that agent security must be approached as a systems problem: the AI model powering the agent must be treated as an untrusted component, and security invariants must be enforced at the system level. Through this lens, efforts to increase model robustness (the dominant viewpoint in the community) are insufficient on their own. Instead, we must complement existing efforts with techniques from the systems security domain. Based on our experience as cybersecurity researchers in operating systems, networks, formal methods, and adversarial machine learning, we articulate a set of core principles, grounded in decades of systems security research, that provide a foundation for designing agentic systems with predictable guarantees. As evidence, we analyze eleven representative real-world attacks on agents and discuss how systems principles, if realized, could have prevented these attacks. We also identify the research challenges that stand in the way of implementing these principles in agents.
