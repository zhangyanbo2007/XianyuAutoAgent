原始摘要与中文对照

中文对照翻译

长期记忆对于专业网络环境中的智能体至关重要，在这些环境中，成功取决于回忆界面可供性、状态动态、工作流和重复出现的故障模式。然而，现有的智能体记忆基准大多关注用户历史、短轨迹或下游任务成功，对于如何直接评估记忆系统是否有效地内化特定环境的经验仍未解决。为了弥补这一空白，我们引入了LongMemEvalV2 (LME-V2)，一个用于评估记忆系统能否帮助智能体在定制环境中获得成为知识渊博的同事所需经验的基准。LME-V2包含451个手动整理的问题，涵盖了网络智能体的五项核心记忆能力：静态状态回忆、动态状态跟踪、工作流知识、环境陷阱和前提意识。这些问题与包含多达500条轨迹和1.15亿个token的历史轨迹配对。我们使用上下文收集公式：记忆系统消耗历史轨迹并返回紧凑的证据以供下游问答。作为这种挑战性设置的初始基线，我们提出了一套两种记忆方法：AgentRunbook-R，一种高效的基于RAG的记忆，具有用于原始状态观察、事件和策略笔记的知识池；以及AgentRunbook-C，它将轨迹存储为文件并调用编码智能体在增强沙盒中收集证据。实验表明，AgentRunbook-C以72.5%的平均准确率取得了最佳性能，优于最强的RAG基线（48.5%）和现成的编码智能体基线（69.3%）。尽管性能提升显著，但基于编码智能体的方法具有高延迟成本。虽然AgentRunbook-C推进了准确性-延迟帕累托前沿，但仍有很大的改进空间。总而言之，这些结果确立了LME-V2作为一个具有挑战性的试验平台，用于开发将累积的智能体轨迹转化为可重用环境经验的长期记忆系统。

原始摘要

Long-term memory is crucial for agents in specialized web environments, where success depends on recalling interface affordances, state dynamics, workflows, and recurring failure modes. However, existing memory benchmarks for agents mostly focus on user histories, short traces, or downstream task success, leaving open how to directly evaluate whether memory systems effectively internalize environment-specific experience. To address this gap, we introduce LongMemEvalV2 (LME-V2), a benchmark for evaluating whether memory systems can help agents acquire the experience needed to become knowledgeable colleagues in customized environments. LME-V2 contains 451 manually curated questions covering five core memory abilities for web agents: static state recall, dynamic state tracking, workflow knowledge, environment gotchas, and premise awareness. These questions are paired with history trajectories containing up to 500 trajectories and 115M tokens. We use a context gathering formulation: memory systems consume history trajectories and return compact evidence for downstream question answering. As initial baselines for this challenging setting, we propose a suite of two memory methods: AgentRunbook-R, an efficient RAG-based memory with knowledge pools for raw state observations, events, and strategy notes, and AgentRunbook-C, which stores trajectories as files and invokes a coding agent to gather evidence in an augmented sandbox. Experiments show that AgentRunbook-C achieves the best performance with 72.5% average accuracy, outperforming the strongest RAG baseline (48.5%) and the off-the-shelf coding agent baseline (69.3%). Despite the strong performance gains, coding agent based methods have high latency costs. While AgentRunbook-C advances the accuracy-latency Pareto frontier, substantial room for improvement remains. Together, these results establish LME-V2 as a challenging testbed for developing long-term memory systems that turn accumulated agent trajectories into reusable environment experience.
