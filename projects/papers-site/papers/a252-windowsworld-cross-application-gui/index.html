原始摘要与中文对照

中文对照翻译

WindowsWorld 是一种以过程为中心的基准测试，旨在评估自主 GUI 代理在专业跨应用程序工作流程中的表现。它包含 181 个任务，分布在 17 个桌面应用程序中，强调多应用程序协调和条件推理，并表明当前计算机使用代理在工作流程跨越多个应用程序和子目标时，性能会急剧下降。

原始摘要

WindowsWorld is a process-centric benchmark for autonomous GUI agents focused on professional cross-application workflows. It contains 181 tasks across 17 desktop applications, emphasizes multi-application coordination and conditional reasoning, and shows that current computer-use agents degrade sharply once workflows span several applications and sub-goals.
