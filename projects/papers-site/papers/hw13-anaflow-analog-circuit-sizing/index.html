核心要点

问题/背景
模拟/混合信号电路设计长期依赖人工经验，设计周期长、仿真开销高，而且很多 AI 自动化方法虽然能优化参数，却缺乏解释性，也很难被设计人员真正接入日常流程。
方法/机制
AnaFlow 提出一个多 agent LLM workflow：不同 agent 分别解释 topology、理解设计目标、推进参数更新，并通过 adaptive simulation strategy 控制仿真预算。它同时保留 human-interpretable reasoning，使 sizing 结果不再只是黑盒最优点，而是可追踪的设计过程。
结果/证据
这篇论文命中了你要求的 `硬件电路设计与仿真` 范围。它的价值不在于某个电路 benchmark，而在于把 reasoning、simulation control 和 explainability 放进同一个 analog design loop，形成更可复用的自动化模式。
收录价值
它不是更高一级，因为验证范围仍然较窄，只在有限电路上证明了 sizing 任务；离完整模拟设计流、版图生成和 silicon-proven flow 还有明显距离。
