问题/背景

方法/机制

原始摘要与中文对照

中文对照翻译

MolmoAct2 是一种完全开放的行动推理模型，旨在用于实际的 VLA（Visual Language Action）部署。它结合了一个新的具身推理 VLM（Vision-Language Model）背骨、大型开放的机器人数据集、一个开放的行动分词器，以及一种自适应深度推理变体，旨在在真实世界控制中保持地基，同时降低延迟。

原始摘要

MolmoAct2 is a fully open action reasoning model for practical VLA deployment. It combines a new embodied-reasoning VLM backbone, large open robot datasets, an open action tokenizer, and an adaptive-depth reasoning variant aimed at maintaining grounding with lower latency in real-world control.
