原始摘要与中文对照

中文对照翻译

评估代理：面向视觉生成模型的有效且可提示的评估框架。视觉生成模型最近的进展使得高质量图像和视频生成成为可能，开辟了多样化的应用。然而，评估这些模型通常需要采样数百或数千张图像或视频，这使得评估过程计算成本高昂，特别是对于固有的慢采样扩散模型。此外，现有的评估方法依赖于僵化的流程，这些流程忽视了特定的用户需求，并且在没有清晰解释的情况下提供数值结果。相比之下，人类只需观察少量样本就能快速形成对模型能力的印象。为了模仿这一点，我们提出了评估代理（Evaluation Agent）框架，该框架采用类人策略进行高效、动态、多轮评估，每轮仅使用少量样本，同时提供详细的、用户定制的分析。它提供了四个主要优势：1) 效率，2) 可根据不同用户需求定制的可提示评估，3) 超越单一数值分数的解释性，以及 4) 跨各种模型和工具的可扩展性。实验表明，评估代理将评估时间缩短至传统方法的10%，同时提供可比较的结果。评估代理框架已完全开源，以推动视觉生成模型及其高效评估的研究。
