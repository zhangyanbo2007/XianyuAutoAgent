完整收录解读

这篇论文研究主动回避学习中，大脑如何把行动带来的安全结果赋予稳定动机价值。

作者发现 PVT→NAc 神经元在成功回避后形成经验依赖的 safety value signal，且该信号随 outcome devaluation 减弱；沉默通路会削弱安全的动机价值但不破坏 action-outcome learning。

机制上，PVT 输入通过 GluA2-lacking AMPA receptor 介导的 PVT-CIN 突触增强调控 NAc 胆碱能中间神经元和多巴胺释放，形成 learned goals 进入动机回路的通道。

它值得收录，因为它给出目标/安全价值如何经丘脑-纹状体通路进入行动选择系统的具体机制，对 agent 的 value assignment、avoidance learning 和内部目标信号路由有明确概念启发。
