原始摘要与中文对照

中文对照翻译

在受监管领域（承保、理赔裁决、临床审查、税务稽查）中，长周期决策代理的企业部署主要采用检索增强型管道，尽管研究文献在过去十年中产生了日益复杂的状态记忆架构。我们认为这反映了一个研究评估尚未揭示的隐藏要求：受监管的部署在四个系统属性上是承载负荷的（确定性重放、可审计的理由、多租户隔离以及用于横向扩展的无状态性）。有状态记忆架构在构建上违反了这些属性，并且随着部署的成熟，违反的程度会加剧。我们提出了确定性投影记忆（DPM），这是一种将代理记忆视为一个仅追加事件日志加上在决策时进行单一任务条件投影的架构。在十个受监管的决策案例和三种记忆预算下，DPM在充足预算下与基于摘要的记忆相匹配（在适度或宽松预算下，四个决策对齐轴上均无统计学显著差异；n=10，配对置换），并且在记忆预算受限时显著优于它：在20倍压缩比下，DPM将事实精确度提高了+0.52（Cohen's h=1.17，p=0.0014），并将推理连贯性提高了+0.53（h=1.13，p=0.0034）。DPM还比有状态基线快7-15倍，因为它在决策时只进行一次LLM调用，而不是在整个轨迹中进行N次调用。一项在温度为零时对每个案例进行10次重放的确定性研究证实，两种情况都继承了残余的API级非确定性，但架构不对称是系统性的：DPM暴露一次非确定性投影调用，而基于摘要的记忆在整个轨迹中暴露N次复合调用。我们最后提出了TAMS，一种用于架构选择的实践者启发式方法，以及对企业运行条件下有状态记忆的故障分析。这项贡献并非新的决策准确性突破。它论证了无状态性是解释企业偏爱较弱但可重放的检索管道的承载负荷属性，并且DPM证明了在不付出检索所带来的决策代价的情况下，可以实现这一属性。

原始摘要

Enterprise deployment of long-horizon decision agents in regulated domains (underwriting, claims adjudication, clinical review, tax examination) is dominated by retrievalaugmented pipelines despite the research literature producing a decade of increasingly sophisticated stateful memory architectures. We argue this reflects a hidden requirement the research evaluation has not surfaced: regulated deployment is load-bearing on four systems properties (deterministic replay, auditable rationale, multi-tenant isolation, and statelessness for horizontal scale). Stateful memory architectures violate these properties by construction, and the margin of violation compounds as deployment matures. We propose Deterministic Projection Memory (DPM), an architecture that treats agent memory as an append-only event log plus a single task-conditioned projection at decision time. On ten regulated decisioning cases at three memory budgets, DPM matches summarization-based memory at generous budgets (no statistically significant difference on any of four decision-alignment axes at moderate or loose budgets; n=10, paired permutation) and substantially outperforms it when memory budget binds: at a 20× compression ratio, DPM improves factual precision by +0.52 (Cohen’s h=1.17, p=0.0014) and reasoning coherence by +0.53 (h=1.13, p=0.0034). DPM is additionally 7–15× faster than the stateful baseline because it makes one LLM call at decision time instead of N calls across the trajectory. A determinism study of 10 replays per case at temperature zero confirms that both conditions inherit residual API-level nondeterminism, but that the architectural asymmetry is systemic: DPM exposes one nondeterministic projection call, summarization-based memory exposes N compounding calls across the trajectory. We conclude with TAMS, a practitioner heuristic for architecture selection, and a failure analysis of stateful memory under enterprise operating conditions. The contribution is not a new decision-accuracy win. It is the argument that statelessness is the load-bearing property explaining enterprise’s preference for weaker but replayable retrieval pipelines, and that DPM demonstrates this property is attainable without the decisioning penalty retrieval pays.
