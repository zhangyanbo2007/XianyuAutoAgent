中文对照翻译

标题：图书馆定理：外部组织如何支配智能体的推理能力 书写通过为扩展的思维链提供一个宽敞的工作空间，改变了人类认知，使推理不可磨灭且广泛传播。随后，随着索引的发明，书写变成了一种检索系统，进一步扩展了其影响力。图书馆和搜索引擎是索引力量的典型例证。基于Transformer的智能体已经通过思维链利用了外部化推理。但结构化检索——对自身推理状态进行索引——仍未得到充分探索。我们将Transformer上下文窗口形式化为I/O页面，并证明配备索引外部记忆的工具增强型智能体比仅限于顺序扫描的智能体实现指数级更低的检索成本：每次查询的页面读取次数为O(logb N ) 对比 Ω(N )，以及在T个推理步骤上的累积成本为O(T logb T ) 对比 Θ(T 2 )。我们在一个受控查找基准上测试了这些预测，该基准涵盖了三种内容类型——随机哈希值、有序整数和百科全书条目——存储大小从50到5,000项不等，并在两个模型代次（GPT-4o-mini和GPT-5.41）中复制了关键条件。在抽象内容（哈希值和整数）上，索引智能体实现了中位数1次页面读取，无论存储大小如何，证实了O(1)的预测。没有索引的排序页面——原则上可以实现二分查找——未能缩小差距：较弱模型无法在大规模下维持二分查找，而较强模型实现了接近最优的log2 N搜索，但仍比索引慢5倍。在熟悉内容（百科全书条目）上，出现了一种竞争性故障模式：模型识别出领域，绕过检索协议，并从参数记忆中生成答案，即使索引结构本身是健全的，也导致灾难性的token消耗。这种参数记忆竞争分离了索引所结合的两种认知操作：理解内容（语言模型擅长之处）和遵循导航协议（当理解诱使它们走捷径时，它们就会失败）。结果主张关注点分离：在语义理解有帮助的地方使用语言模型进行索引构建，而在语义理解有害的地方使用确定性算法进行索引遍历。

原始摘要

Writing transformed human cognition by providing a capacious workspace for extended chains of thought, making reasoning indelible and widely communicable. Then, by the invention of the index, writing became a retrieval system, extending its reach even further. The library and the search engine are canonical demonstrations of index power. Externalized reasoning is already exploited by transformer-based agents through chain-of-thought. But structured retrieval—indexing over one’s own reasoning state—remains underexplored. We formalize the transformer context window as an I/O page and prove that toolaugmented agents with indexed external memory achieve exponentially lower retrieval cost than agents restricted to sequential scanning: O(logb N ) versus Ω(N ) page reads per query, and O(T logb T ) versus Θ(T 2 ) cumulative cost over T reasoning steps. We test these predictions on a controlled lookup benchmark across three content types—random hashes, ordered integers, and encyclopedia entries—varying store size from 50 to 5,000 items, and replicate key conditions across two model generations (GPT-4o-mini and GPT-5.41 ). On abstract content (hashes and integers), the indexed agent achieves median 1 page read regardless of store size, confirming the O(1) prediction. Sorted pages without an index— which in principle enable binary search—fail to close the gap: the weaker model cannot sustain binary search at scale, and the stronger model achieves near-optimal log2 N search but still loses to the index by 5×. On familiar content (encyclopedia entries), a competing failure mode emerges: the model recognizes the domain, bypasses the retrieval protocol, and generates answers from parametric memory, producing catastrophic token expenditure even when the index structure itself is sound. This parametric memory competition dissociates the two cognitive operations that indexing combines: understanding content (where language models excel) and following navigational protocols (where they fail when understanding tempts them to shortcut). The result argues for a separation of concerns: use language models for index construction, where semantic understanding helps, and deterministic algorithms for index traversal, where it hurts.
