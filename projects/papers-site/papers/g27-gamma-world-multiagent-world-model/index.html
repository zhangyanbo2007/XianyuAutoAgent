核心要点

问题/背景
交互式视频 world models 过去多默认单一控制信号，但游戏、机器人和多主体仿真都需要多个 agent 在同一环境中同时行动。
方法/机制
Gamma-World 的核心是把多 agent identity 和交互建模进生成式 world model：Simplex Rotary Agent Encoding 让 agent 既可区分又保持置换对称，Sparse Hub Attention 用 hub token 降低跨 agent attention 成本。
结果/证据
为了实时交互，论文还把 full-context diffusion teacher 蒸馏为带 KV cache 的 causal student，实现 action-responsive rollout。
收录价值

完整收录解读

交互式视频 world models 过去多默认单一控制信号，但游戏、机器人和多主体仿真都需要多个 agent 在同一环境中同时行动。

Gamma-World 的核心是把多 agent identity 和交互建模进生成式 world model：Simplex Rotary Agent Encoding 让 agent 既可区分又保持置换对称，Sparse Hub Attention 用 hub token 降低跨 agent attention 成本。

原始摘要与中文对照

中文对照翻译

Gamma-World 引入了一种生成式多智能体世界模型，用于交互式模拟，采用辛普克斯旋转智能体编码实现排列等价的智能体身份，使用稀疏环状注意力机制进行高效的跨智能体交互，并利用因果蒸馏实现24帧每秒的动作响应式滚播。

原始摘要

Gamma-World introduces a generative multi-agent world model for interactive simulation, with Simplex Rotary Agent Encoding for permutation-equivalent agent identity, Sparse Hub Attention for efficient cross-agent interaction, and causal distillation for 24 FPS action-responsive rollout.
