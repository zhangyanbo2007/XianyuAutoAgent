原始摘要与中文对照

中文对照翻译

视觉皮层中基于动态上下文的对象表征更新 Giacomo Aldegheri1,2,3*, Surya Gayet1,4, Marius V. Peelen1 真实世界场景中的物体通常可见性差或部分可见，例如因为它们被遮挡或出现在视野边缘。真实世界视觉的另一个挑战是其动态性，导致物体（例如，它们的尺寸和方向）的外观随着我们的移动而变化。然而，值得注意的是，这些变化可以从周围场景的三维结构中预测。在两项功能性磁共振成像研究中，我们发现视觉皮层利用这种预测性上下文信息动态更新对象表征。首先，当物体与周围场景同步（而非不同步）旋转时，视觉皮层对物体的表征得到增强。其次，即使物体被完全遮挡，也可以从视觉皮层活动中解码出推断的物体方向。这些发现表明，视觉皮层中的预测过程遵循环境的几何结构，为支持动态自然视觉中的物体感知提供了一种机制。

原始摘要

Dynamic context–based updating of object representations in the visual cortex Giacomo Aldegheri1,2,3*, Surya Gayet1,4, Marius V. Peelen1 Objects in real-­world scenes are often poorly or partially visible, for example because they are occluded or appear in the periphery. An additional challenge of real-­world vision is that it is dynamic, causing the appearance of objects (e.g., their size and orientation) to change as we move. Notably, however, these changes are predictable from the three-­dimensional structure of the surrounding scene. In two functional magnetic resonance imaging studies, we find that the visual cortex dynamically updates object representations using this predictive contextual information. First, visual cortical representations of objects were enhanced when they rotated congruently (versus incongruently) with the surrounding scene. Second, the inferred orientation of the object could be decoded from visual cortex activity even when the object was fully occluded. These findings indicate that predictive processes in the visual cortex follow the geometric structure of the environment, providing a mechanism to support object perception in dynamic natural vision.
