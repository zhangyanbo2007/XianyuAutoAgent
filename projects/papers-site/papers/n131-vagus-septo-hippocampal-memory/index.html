原始摘要与中文对照

中文对照翻译

这篇《自然-通讯》论文识别出一条迷走神经介导的隔-海马乙酰胆碱通路，通过该通路，营养物质摄入可增强海马依赖性记忆，并表明迷走神经切断术、隔区胆碱能消融或西式饮食暴露会破坏这种记忆支持。

原始摘要

This Nature Communications paper identifies a vagus-nerve-mediated septo-hippocampal acetylcholine pathway by which nutrient consumption enhances hippocampal-dependent memory, and shows that vagotomy, septal cholinergic ablation, or Western diet exposure disrupts this memory support.
