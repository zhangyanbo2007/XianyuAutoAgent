完整收录解读

问题与背景

Cortisol is known to enhance memory for salient and emotionally charged experiences, but the brain-wide network mechanisms responsible for this effect have remained unclear.

方法/新意

The paper combines pharmacological fMRI with dynamic connectome-based predictive modeling to identify high-temporal-resolution whole-brain networks that predict arousal and subsequent memory under cortisol.

意义/放在仓库中的位置

This belongs on the cognitive neuroscience mainline as a strong dynamic-network paper on memory formation. It is particularly useful for the repository because it emphasizes temporally resolved whole-brain mechanisms rather than only static localization.

局限/为何不更高

The work is a strong domain contribution, but its methodological and conceptual impact is still mostly contained within affective memory neuroscience, so breakthrough is appropriate.
