原始摘要与中文对照

中文对照翻译

ClawGUI 是一种开源的全栈框架，用于 GUI 代理，涵盖强化学习训练、标准化的评估以及实机部署。它解决了 GUI 代理工作中的基础设施差距：不稳定的在线环境、漂移的评估协议以及糟糕的部署路径。ClawGUI-RL 支持并行虚拟环境和物理设备；ClawGUI-Eval 标准化了六个基准测试；ClawGUI-Agent 在 Android、HarmonyOS 和 iOS 上部署训练好的代理，并采用混合 CLI-GUI 控制和持久内存。

原始摘要

ClawGUI is an open-source full-stack framework for GUI agents, covering RL training, standardized evaluation, and real-device deployment. It addresses infrastructure gaps in GUI-agent work: unstable online environments, drifting evaluation protocols, and poor deployment paths. ClawGUI-RL supports parallel virtual environments and physical devices; ClawGUI-Eval standardizes six benchmarks; ClawGUI-Agent deploys trained agents across Android, HarmonyOS, and iOS with hybrid CLI-GUI control and persistent memory.
